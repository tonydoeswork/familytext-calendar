# FamilyText Calendar

**Anthony Anderson**

FamilyText Calendar is a private household SMS application that allows registered family members to manage a shared calendar entirely through text messages.

This is a personal application operated by Anthony Anderson for household use only. It is not a commercial product and is not available to the general public.

---

## How It Works

Registered users send natural language text messages to a dedicated phone number. The application processes the message and adds events to, or retrieves events from, a shared family Google Calendar.

Examples:
- "Dentist Thursday 2pm" → adds event to calendar
- "What's today?" → returns today's schedule
- "HELP" → returns usage guide
